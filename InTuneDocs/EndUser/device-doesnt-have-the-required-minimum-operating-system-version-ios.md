# El dispositivo no tiene la versión necesaria mínima del sistema operativo

El dispositivo iOS no cumple con la versión mínima necesaria del sistema operativo requerida por el departamento de TI.  Antes de actualizar el dispositivo, compruebe la versión del sistema operativo actual; para ello, pulse **Configuración** &gt; **Acerca de**. Para actualizar el dispositivo a la versión más reciente, pulse **Configuración** &gt; **General** &gt; **Actualización de software**..
